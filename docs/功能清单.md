# 功能清单

> 本文档与 `TODO.md` 配套使用，TODO.md 规划**什么时候做**，本文档定义**做什么**。

---

## 1. 基础数据

| 模块 | 功能点 | 说明 |
|------|--------|------|
| 商品管理 | 分类管理 | 树形分类，支持多级 |
| 商品管理 | 商品 CRUD | SKU、条码、名称、单位、规格、售价、成本价、预警库存 |
| 商品管理 | 条码/拼音码搜索 | 扫条码或输入首字母快速定位商品 |
| 商品管理 | Excel 导入 | 批量导入商品资料，用于初始化或批量更新 |
| 往来单位 | 客户/供应商管理 | 统一管理，type 字段区分角色，一个人可同时是客户和供应商 |
| 仓库管理 | 仓库 CRUD | 支持实体仓和虚拟仓 |

## 2. 采购管理

| 模块 | 功能点 | 说明 |
|------|--------|------|
| 采购订单 | 订单 CRUD | 创建、编辑、审核、关闭，状态机流转 |
| 采购订单 | 订单查询 | 按单号、供应商、日期范围、状态筛选 |
| 采购入库 | 入库单 | 支持分批次入库，一张采购订单可分多次入库 |
| 采购入库 | 自动记账 | 入库完成自动增加库存、写入流水、生成应付暂估 |
| 采购退货 | 退货出库 | 退回给供应商，库存扣减+流水记录，冲减应付 |
| 供应商对账 | 对账汇总 | 按单或按期间汇总应付款 |

## 3. 销售管理

| 模块 | 功能点 | 说明 |
|------|--------|------|
| 报价单 | 报价管理 | 可选功能，小商家可直接跳过到销售订单 |
| 销售订单 | 订单 CRUD | 创建、审核、出库、关闭，状态机流转 |
| 销售订单 | 订单查询 | 按单号、客户、日期、状态筛选 |
| 销售出库 | 出库单 | 支持多次出库，一张订单可分多次发货 |
| 销售出库 | 自动记账 | 出库完成自动扣减库存、写入流水、生成应收账款 |
| 销售退货 | 退货入库 | 客户退货→验货→入库→退款，库存回滚+流水，冲减原毛利 |
| 小票打印 | 热敏打印 | 支持 58mm/80mm 热敏打印机，销售即打 |

## 4. 库存管理

| 模块 | 功能点 | 说明 |
|------|--------|------|
| 实时库存 | 库存查询 | 按商品/按仓库/按全部维度查询当前库存 |
| 库存流水 | 流水追溯 | 每一笔数量变动的来龙去脉可查，只读不可改 |
| 盘点 | 盘点流程 | 盘点单→录入实盘数→差异计算→审核→自动调账 |
| 库存预警 | 预警通知 | 低于安全库存时在看板提示，支持设置预警阈值 |

## 5. 会员管理

| 模块 | 功能点 | 说明 |
|------|--------|------|
| 会员资料 | 基本信息 | 姓名、电话、等级、开卡日期 |
| 会员价 | 价格体系 | 不同等级会员享受不同售价（商品维度设置） |
| 消费记录 | 历史查询 | 按会员查看历史购买记录 |
| 积分/储值 | 扩展功能 | MVP 阶段可选，上线后迭代 |

## 6. 财务管理

| 模块 | 功能点 | 说明 |
|------|--------|------|
| 收款管理 | 收款核销 | 核销应收账款，支持微信/支付宝/现金/银行卡 |
| 付款管理 | 付款核销 | 核销应付账款 |
| 资金流水 | 流水时间线 | 所有收付款按时间排列，按支付方式汇总 |
| 应收账款表 | 欠款统计 | 谁欠了多少，超期提醒（30/60/90 天区间） |
| 应付账款表 | 欠款统计 | 欠供应商多少 |
| 利润报表 | 毛利计算 | 按商品/按客户/按时间段统计，公式：收入-成本（移动加权平均） |

## 7. 系统管理

| 模块 | 功能点 | 说明 |
|------|--------|------|
| 账号权限 | RBAC | 基于角色的权限控制，支持多账号 |
| 数据字典 | 字典管理 | 单据类型、支付方式、订单状态等枚举统一管理 |
| 系统参数 | 参数配置 | 预警阈值、小数精度、默认税率 |
| 操作日志 | 审计日志 | 关键操作记录操作人、时间、内容 |

## 8. 看板与报表

| 模块 | 功能点 | 说明 |
|------|--------|------|
| 每日看板 | Dashboard | 今日营业额、今日毛利、应收/已收、库存预警、快捷入口 |
| 利润分析 | 报表 | 按商品/客户/时间段的毛利趋势图（ECharts） |
